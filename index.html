Breadcrumbs

Breadcrumb navigation, also known as a breadcrumb trail, is a graphical user interface (GUI) element that allows users to keep track of their current location on a website or application. Consist of a horizontal row of clickable links.

  • Are usualy displayed as a horizontal list of pages.
  • Are placed in the header, left-aligned and below any primary navigation.
  • Are separated with an ">" or an "/".
  • Shop
  • Groceries
  • Strawberries

HTML

<ul class="breadcrumb">
  <li>
    <a href= "#">Shop</a>
  </li>
  <li>
    <a href= "#">Groceries</a>
  </li>
  <li>
    <a href= "#">Strawberries</a>
  </li>
</ul>

CSS

ul.breadcrumb li {
  display: inline;
}
ul.breadcrumb li + li::before {
  content: ">";
  padding-right: 0.4rem;
}
ul.breadcrumb li + li::after {
  padding-left: 0.4rem;
}
ul.breadcrumb li a {
  text-decoration: none;
  color: #3c3491;
}
ul.breadcrumb li a:hover {
  cursor: pointer;
  color: #fcb81a;
}
